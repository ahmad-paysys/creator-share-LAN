# Security Policy

## Supported Versions

This project currently supports the latest main branch only.

## Reporting a Vulnerability

Please avoid opening a public issue for security vulnerabilities.

1. Send a private report with reproduction steps and impact.
2. Include affected endpoints and example payloads.
3. Allow reasonable time for triage and patching before disclosure.

## Hardening Notes

- This app is intended for trusted LAN environments.
- No authentication is included by design.
- Keep EXPOSE_TO_LAN disabled for untrusted networks.
